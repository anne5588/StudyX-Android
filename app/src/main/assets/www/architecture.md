## 🔧 技术架构

### 当前Web版本
- 纯前端：HTML + CSS + JS
- 数据存储：localStorage
- 可打包为：WebView APP

### APP化方案

#### 方案A：WebView封装（推荐）
```
工具：Cordova / Capacitor
优点：
- 现有代码直接可用
- 1-2天即可完成打包
- 支持离线运行
- 可访问本地文件系统（用于导入词汇）

打包步骤：
1. 安装Capacitor: npm install @capacitor/core @capacitor/cli
2. 初始化: npx cap init StudyX com.studyx.app
3. 构建: npx cap add android
4. 打包: npx cap open android → Android Studio生成APK
```

#### 方案B：PWA（渐进式Web应用）
```
优点：无需打包，添加到主屏幕即可
缺点：部分安卓机型支持不佳
```
