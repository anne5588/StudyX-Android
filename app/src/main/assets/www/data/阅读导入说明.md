# StudyX 英语阅读导入说明

## 📖 支持两种导入格式

### 格式1：单篇文章导入（推荐新手）

适合一次导入一篇阅读文章：

```json
{
  "year": "2024",
  "title": "文章英文标题",
  "source": "2024年申硕英语真题 Passage 1",
  "sentences": [
    {
      "en": "英文句子1",
      "zh": "中文翻译1"
    },
    {
      "en": "英文句子2",
      "zh": "中文翻译2"
    }
  ],
  "questions": [
    {
      "q": "问题",
      "options": ["A. 选项1", "B. 选项2", "C. 选项3", "D. 选项4"],
      "answer": "B",
      "analysis": "解析说明"
    }
  ]
}
```

### 格式2：批量导入（推荐整理多篇）

适合一次导入多篇阅读文章：

```json
{
  "articles": [
    {
      "year": "2024",
      "title": "文章1标题",
      "source": "2024年真题 Passage 1",
      "sentences": [...],
      "questions": [...]
    },
    {
      "year": "2023",
      "title": "文章2标题",
      "source": "2023年真题 Passage 2",
      "sentences": [...],
      "questions": [...]
    }
  ]
}
```

## 📝 字段说明

### 必填字段

| 字段 | 说明 | 示例 |
|------|------|------|
| `year` | 年份 | `"2024"`、`"2023"` |
| `title` | 文章英文标题 | `"The Impact of Technology"` |
| `source` | 来源说明 | `"2024年申硕英语真题 Passage 1"` |

### 选填字段

| 字段 | 说明 | 示例 |
|------|------|------|
| `questions` | 阅读理解题目 | 见下方详细说明 |

## ✂️ 句子分割建议

### 分割原则

1. **按意群分割**：不要把太长的段落作为一个句子
2. **一句一意**：每句英文对应一个完整的中文翻译
3. **便于阅读**：每行不要太长，适合手机屏幕阅读

### 好的例子

```json
{
  "en": "Technology has transformed the way we learn and teach in the 21st century.",
  "zh": "科技已经改变了21世纪我们学习和教学的方式。"
}
```

### 避免的例子

❌ 太长的一整段：
```json
{
  "en": "Technology has transformed the way we learn and teach in the 21st century. Online platforms allow students to access educational resources from anywhere in the world. However, some experts argue that excessive screen time may affect students' physical health.",
  "zh": "科技已经改变了21世纪我们学习和教学的方式。在线平台让学生可以从世界任何地方获取教育资源。然而，一些专家认为过度的屏幕时间可能会影响学生的身体健康。"
}
```

✅ 应该分割成三句：
```json
[
  { "en": "Technology has transformed the way we learn and teach in the 21st century.", "zh": "科技已经改变了21世纪我们学习和教学的方式。" },
  { "en": "Online platforms allow students to access educational resources from anywhere in the world.", "zh": "在线平台让学生可以从世界任何地方获取教育资源。" },
  { "en": "However, some experts argue that excessive screen time may affect students' physical health.", "zh": "然而，一些专家认为过度的屏幕时间可能会影响学生的身体健康。" }
]
```

## 📋 真题整理步骤

### Step 1: 获取真题文字

从你的PDF中提取英文原文和中文翻译：

1. **OCR识别**：使用Adobe Acrobat、ABBYY FineReader等工具识别PDF文字
2. **复制粘贴**：如果PDF是文字版，直接复制
3. **拍照识别**：用手机拍照，使用微信/QQ的文字识别功能

### Step 2: 翻译整理

将英文分割成句子，并添加中文翻译：

```
原文：
Technology has transformed the way we learn and teach. Online platforms allow students to access resources from anywhere.

整理后：
1. Technology has transformed the way we learn and teach.
   → 科技已经改变了我们学习和教学的方式。
   
2. Online platforms allow students to access resources from anywhere.
   → 在线平台让学生可以从任何地方获取资源。
```

### Step 3: 编写JSON

按照格式要求编写JSON文件：

```json
{
  "year": "2024",
  "title": "Technology and Education",
  "source": "2024年申硕英语真题 Passage 1",
  "sentences": [
    {
      "en": "Technology has transformed the way we learn and teach.",
      "zh": "科技已经改变了我们学习和教学的方式。"
    },
    {
      "en": "Online platforms allow students to access resources from anywhere.",
      "zh": "在线平台让学生可以从任何地方获取资源。"
    }
  ]
}
```

### Step 4: 导入APP

1. 保存为 `.json` 文件
2. 在APP阅读页面点击"**导入**"
3. 选择JSON文件
4. 完成！

## 💡 快速制作技巧

### 使用Excel整理，然后转换

1. 在Excel中创建三列：序号 | 英文 | 中文
2. 逐行填入内容
3. 使用在线工具将Excel转为JSON
4. 按照格式微调即可

### 使用AI辅助翻译

1. 将英文原文输入ChatGPT/Claude
2. 提示词："请将以下英文按句子分割，并提供准确的中文翻译，格式为JSON"
3. 复制结果，稍作调整即可使用

## ⚠️ 常见问题

### Q1: 导入失败怎么办？

检查以下几点：
- 文件格式是否为 `.json`
- 是否包含必填字段（year, title, source, sentences）
- 句子数组是否为空
- JSON格式是否正确（可使用在线JSON校验工具）

### Q2: 可以导入没有翻译的文章吗？

不可以。每句英文必须有对应的中文翻译（`zh`字段），这是点读功能的基础。

### Q3: 可以导入只有文章没有题目的吗？

可以。`questions`字段是选填的，没有题目也能正常阅读。

### Q4: 如何修改已导入的文章？

目前需要：
1. 导出所有文章
2. 在JSON文件中修改
3. 删除APP内原有文章（清空数据）
4. 重新导入修改后的文件
